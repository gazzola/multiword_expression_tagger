## Program
The program (``src/main.py`` module) identifies multiword expressions of time using the patterns 1 and 2 explained [here](https://github.com/RedHenLab/NLP/issues/1).
#### Usage:
``python main.py``

#### Output:

- time fly
- as seconds go by
